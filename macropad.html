Macropad

Summary: As a new hire at Texas Inventionworks, I was tasked with creating a macropad. I was extremely excited to start this project since I had already planned on creating a macropad for personal use and had sketches ready. This expedited the design process and motivated me to make this macropad suitable for personal use.

Problem Statement: Create a macropad capable of typing to pass two tests and a competition: the first test is to type a predetermined string, the second test is to type a surprise string, and the competition is to create the most "attractive" macropad.

Project Timeline: One month will be given to complete this project with a progress check in the middle. My personal goal was to have the CAD and electronics done, so I could spend the remaining time working on the aesthetics.

Design Process: For this project, I wanted to create something that looked a bit different than what was on the market. I wanted to make the footprint as small as possible to minimize desk space, and create an almost floating effect with the keys. We were also given a Raspberry Pi Pico, so I made sure to include space in the CAD for an adapter to switch the Micro-USB to USB-C which I purchased for around a dollar on AliExpress, my source for a lot of the electronics I use. For the material, I opted to use PETG for the plate and a very tight press fit. For the keys, I used resin from the SLA printers at Texas Inventionworks that would retain high dimensional accuracy for the connections between the keys and the switches. The main body was printed from PLA to accommodate the heated inserts I used to secure the plate.

Future Considerations: This project was extremely interesting, and I hope to further this design in the future by adding more features. I also hope to create a smaller main body footprint that would make the design even sleeker. Additionally, for the competition, I had overlooked how people enjoyed uniqueness. This led me to lose the competition despite creating something that may have been more practical.

Skills: CAD, Design for Assembly (DFA), Electronics Assembly, and Product Design
